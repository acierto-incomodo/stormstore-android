StormStore es la aplicación oficial para dispositivos Android que te permite acceder a nuestra tienda de juegos y aplicaciones de forma rápida y fluida.

## ✨ Características Principales

- **Experiencia Nativa**: Desarrollada con **Jetpack Compose** y **Material Design 3** para una interfaz moderna y adaptativa.
- **Navegación Fluida**: WebView optimizado con manejo de caché y almacenamiento DOM para una experiencia web similar a una app nativa.
- **Sistema de Actualizaciones**: Verificación automática de versiones mediante la API de GitHub Releases. Nunca te quedarás atrás.
- **Gestor de Descargas**: Descarga e instala aplicaciones (APKs) directamente desde la tienda con integración de `DownloadManager`.
- **Resiliencia**: Pantallas de carga personalizadas y gestión de errores de conexión con opción de reintento.
- **Soporte Edge-to-Edge**: Aprovecha todo el tamaño de pantalla de los dispositivos modernos.

## 🛠️ Tecnologías Utilizadas

- **Kotlin**: Lenguaje principal de desarrollo.
- **Jetpack Compose**: Para la construcción de la interfaz de usuario.
- **Material 3**: El sistema de diseño más reciente de Google.
- **WebView**: Para la renderización del contenido de la tienda.
- **GitHub API**: Para el control de versiones y actualizaciones.

## 🛡️ Permisos Requeridos

Para el correcto funcionamiento, la aplicación solicitará:
- **Notificaciones**: Para informarte sobre el estado de tus descargas.
- **Instalar aplicaciones desconocidas**: Necesario para poder instalar los APKs que descargues desde StormStore.
